# Fonts

All fonts are loaded via Google Fonts CDN (see `colors_and_type.css`). No local TTFs are bundled.

| Role | Family | Source |
|---|---|---|
| Display | **Outfit** | Google Fonts — variable, geometric, rounded, energetic |
| Body | **DM Sans** | Google Fonts — friendly, clean, opsz variable |
| Mono | **JetBrains Mono** | Google Fonts — strong tabular numerals for stats & code |

**Substitution flag:** No Amanotes-specific brand font was provided. These three are open-source substitutes that fit the "youthful + energetic + slightly nerdy" Ama Duo Data vibe. Swap in the real Amanotes display face when available — `--font-display` in `colors_and_type.css` is the only place that needs updating.
