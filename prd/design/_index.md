## Principles

1. **Internal pro tool, not a kid's game.** Senior staff use this. Playful, never infantile.
2. **Amanotes brand.** Reuse brand colors and typography from the existing design system; add DuoData-specific elements (level badges, league art) at the periphery.
3. **Type-led, illustration-light.** Visual hierarchy is type + spacing + restrained color. Illustrations only on celebration moments.
4. **One screen, one job.** Lesson cards show one prompt + one action. No sidebar clutter, no upsell.
