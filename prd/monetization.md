# Monetization

**N/A — DuoData is an internal Amanotes tool.**

No ads, no IAP, no subscription. Funded by GAMA Operating Budget under the DnI line item.

## Why this section exists

The PRD template defaults to a hybrid-casual monetization model. That model does not apply here. This file is kept to satisfy the spec contract and to make the decision explicit, not to define monetization.

## What this means for features

- Feature files do **not** include a "Monetization touchpoints" section.
- Ad-format references in content (RW, FS, IS, BN, AOA, MREC) appear **only** as lesson subject matter — teaching Amanoters about Amanotes products' monetization — and never as actual monetization mechanisms inside DuoData itself.
- Subscription / RevenueCat / Balancy / Metica / Nefta are likewise discussed as lesson topics, never integrated.

## If this changes

Adding monetization to DuoData would require an ADR. Likely never; if it ever becomes a paid product for external partners (e.g., MEP licensees), that is a separate product launch, not a config flag.
